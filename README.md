# OK Irrigation — Website Preview

**▶ Live preview:** https://elitanz.github.io/ok-irrigation-preview/
**▶ SEO cleanup list:** https://elitanz.github.io/ok-irrigation-preview/seo-report.html

A concept redesign for **OK Irrigation Services LLC** (Chanhassen, MN) — a clean,
mobile-friendly homepage built from the business's real public details (services,
phone, service area, and 5.0 Angi rating).

This preview also fixes the six SEO gaps found on the current live site:
a real page title, a meta description, keyword-rich headings, local service/area
copy, `LocalBusiness` structured data (JSON-LD), and visible star ratings.
